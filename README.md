# Meal Planning MVP

## Project Description
The Meal Planning MVP is a web application designed to help users plan their meals for the week. It offers features to simplify meal planning and provide recipe suggestions based on user preferences.

## Features
- **User Authentication:** Secure login for users to save their preferences.
- **Meal Planning:** Customizable meal calendars where users can add and view their planned meals.
- **Recipe Suggestions:** Generate recipe ideas based on selected ingredients.
- **Shopping List:** Automatically create a shopping list based on planned meals.
- **Responsive Design:** Works on both desktop and mobile devices.

## Setup Instructions
1. **Clone the Repository:**  
   `git clone https://github.com/naphekiet-gif/Meal-planning-mvp.git`

2. **Navigate to the Directory:**  
   `cd Meal-planning-mvp`

3. **Install Dependencies:**  
   Run `npm install` to install the required packages.

4. **Start the Application:**  
   Use `npm start` to launch the application in your browser.

## Usage
- Upon launching the application, create an account or log in.
- Use the meal planner to select your meals and generate a shopping list.
- Explore various recipes and customize your meal plans based on your dietary preferences.
